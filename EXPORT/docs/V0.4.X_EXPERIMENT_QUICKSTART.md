**Calculate GFLOPS:**
```
For NxN matrix multiply:
operations = 2 * N^3
GFLOPS = operations / time / 1e9

Example: N=1000, time=0.5s
GFLOPS = 2 * 1000^3 / 0.5 / 1e9 = 4.0
```
